First, open the cmd twice, one for Server and last for Client.
Then in first cmd type `java Server` and hit the enter
In another opened cmd type `java Client` and hit the enter
And to exchange the messages with another device under same wifi connection which has the same IP address
If you're not under the same wifi then use the public IP address which can get by typing the 'What is my ip address' in web browser
Lastly, when you get your ip address then `private static final String SERVER_IP = "localhost";` replace your ip address in localhost
then run that edited code in the device you want to exchange messages
